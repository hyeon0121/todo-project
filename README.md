# todo-project

> A Vue.js + Spring boot project

### Frontend build Setup

 1. npm 설치
 ``` bash
 yum install npm

 ```
 
 2. 프로젝트 빌드
``` bash
# frontend 폴더로 이동
cd fe

# package 설치
npm install

# module 추가
npm --save install vue bootstrap bootstrap-vue vue-datetime vue-datetime-picker vue-star-rating moment vue-momentjs axios luxon

# build
npm run build

# http://localhost:8888 로 접속
npm run dev

```
